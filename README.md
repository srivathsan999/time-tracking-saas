# TimeFlow - Time Tracking SaaS Template

## Overview
TimeFlow is a premium, high-quality HTML5 SaaS template designed for Time Tracking software. It features a modern design system with Light/Dark mode, Glassmorphism elements, and smooth GSAP animations.

## Directory Structure
- `assets/` - Contains all static assets
  - `css/` - Stylesheets (Bootstrap grid, Style, Dark Mode, Animations)
  - `js/` - JavaScript files (Main, Theme Toggle, Animations)
  - `images/` - Locally stored images
- `*.html` - All site pages

## Features
- **Strictly Typed Design System**: Using CSS variables for easy theming.
- **Dark Mode**: Fully supported with persistent state.
- **GSAP Animations**: Fade-ups, staggers, and smooth interactions.
- **Responsive**: Built on Bootstrap 5 Grid.
- **No Dependencies**: No React, Vue, or jQuery. Pure Vanilla JS.

## Setup
Simply open `index.html` in your browser. No build process required. 
Ensure you have an internet connection for the Google Fonts (Inter). Bootstrap and GSAP are included in the assets folder (if downloaded) or linked via CDNs if text-based.
*Note: In this delivery, libraries are downloaded to `assets/`.*
